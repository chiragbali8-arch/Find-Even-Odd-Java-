# Find-Even-Odd-Java-
A simple java console program that checks whether the number is even or odd using loops , conditional statements and user input.

# Features 

Takes user input using Scanner.

Checks Whether a number is even or odd .

Prevents input of 0.

Allow user to continue checking.

Allow user to quit program.

# Concepts Used 

java 

Scanner Class

whole loop

if-else Statements

String Comparison

break Statement
